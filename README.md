# 📚 Tobeton - Smart Study App with AI Companion

**Tobeton** is an intelligent study application designed to help students learn more effectively. Developed by a team of 4 developers, Tobeton features an AI companion named **Tobe** that assists with flashcards, quizzes, summaries, and more.
